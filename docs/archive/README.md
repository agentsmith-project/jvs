# Archive Reading Guide

**Status:** Archived, non-release-facing, and not part of the v0 public contract.

This directory is a navigation marker for historical material. Files are not
moved here automatically because tests, links, and release evidence may still
refer to their original paths.

Use this rule when reading older docs:

- Active public UX is the save point contract: folder, workspace, save point,
  save, history, view, restore, recovery, and cleanup.
- Storage and package names in older docs are implementation facts or
  historical facts only. They are not public commands, selectors, examples, or
  workflow concepts.

Before promoting any archived or historical behavior back into the public
contract, update `../02_CLI_SPEC.md`, `../06_RESTORE_SPEC.md`,
`../PRODUCT_PLAN.md`, `../ARCHITECTURE.md`, the release policy, and
conformance coverage together.
