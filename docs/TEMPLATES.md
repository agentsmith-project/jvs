# JVS Templates Pointer

**Status:** Archived, non-release-facing, and not part of the v0 public contract.

This page is not GA guidance and does not define supported template files,
path-selection rules, or partial-save behavior.

Use the GA user documents instead:

- [Quickstart](user/quickstart.md)
- [Examples](user/examples.md)
- [Safety](user/safety.md)
- [Command reference](user/commands.md)

GA workflows choose the folder boundary before `jvs init`, create save
points with `jvs save -m`, read history with `jvs history` and `jvs view`,
restore with `jvs restore`, and use `jvs doctor` or `jvs recovery` for health
and interrupted-restore handling.
